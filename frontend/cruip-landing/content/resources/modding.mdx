## 🎨 使用指南

### 导入到项目

import { Steps } from 'nextra/components'

<Steps>

### 下载资源

从下载中心获取需要的资源包。

### 解压文件

解压到临时目录。

### 分类整理

将资源复制到项目对应目录：
- 立绘 → `figure/`
- 背景 → `background/`
- UI → `ui/`
- 音频 → `bgm/` 或 `vocal/`

### 在脚本中引用

使用资源文件名引用。

</Steps>

### 文件命名规范

```
# 角色立绘
character_emotion_costume.png
例: tomori_happy_school.png

# 背景
location_time_weather.jpg
例: classroom_day_sunny.jpg

# UI 元素
ui_type_state.png
例: button_start_normal.png
```

## 🔧 自定义修改

### 使用 Photoshop

1. 打开 PSD 源文件
2. 修改图层内容
3. 调整颜色/效果
4. 导出为 PNG

### 使用 Bandori Craft

1. 导入资源到 Bandori Craft
2. 使用内置编辑器修改
3. 应用滤镜和效果
4. 导出优化后的资源

### 批量处理

使用 AnoGO 工具批量处理：

```bash
# 批量调整尺寸
anogo resize --width=1920 --height=1080 *.jpg

# 批量转换格式
anogo convert --to=webp --quality=90 *.png

# 批量添加水印
anogo watermark --image=logo.png --position=bottom-right *.png
```

## 💡 最佳实践

<Callout type="info">
  **优化建议**
  - 使用 WebP 格式减小文件大小
  - 立绘使用 PNG 保持透明度
  - 背景可使用 JPG 格式
  - 音频使用 MP3 192kbps 平衡质量与大小
</Callout>

<Callout type="warning">
  **注意事项**
  - 保持原始文件备份
  - 测试修改后的资源
  - 注意文件命名规范
  - 遵守版权协议
</Callout>

## 📊 资源规格

### 图片规格

| 类型 | 推荐尺寸 | 格式 | 备注 |
|------|---------|------|------|
| 角色立绘 | 高度 1500-2000px | PNG | 透明背景 |
| 背景图片 | 1920x1080 | JPG/WebP | 高质量 |
| UI 元素 | 根据设计 | PNG | 透明背景 |
| 图标 | 64x64 或 128x128 | PNG | 透明背景 |

### 音频规格

| 类型 | 格式 | 码率 | 备注 |
|------|------|------|------|
| BGM | MP3/OGG | 192kbps | 循环播放 |

## 🆘 常见问题

### 资源无法显示

- 检查文件路径
- 确认文件格式
- 验证文件完整性
- 查看控制台错误

### 图片质量问题

- 使用高质量源文件
- 避免过度压缩
- 保持合适分辨率
- 使用无损格式

### 音频播放问题

- 检查音频格式
- 确认浏览器支持
- 测试音频文件
- 查看错误日志
