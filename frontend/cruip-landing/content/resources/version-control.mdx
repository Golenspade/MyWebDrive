### 升级步骤

#### 引擎升级

1. **下载新版本**
   - 从下载中心获取最新版本

2. **备份旧版本**
   ```bash
   cp -r WebGAL_old WebGAL_backup
   ```

3. **安装新版本**
   - 解压新版本到新目录
   - 或覆盖旧版本（已备份）

4. **迁移项目**
   - 复制项目文件到新版本
   - 检查配置文件
   - 测试运行

5. **验证功能**
   - 测试所有功能
   - 检查资源加载
   - 验证脚本运行

#### 工具升级

1. **卸载旧版本**（可选）
2. **安装新版本**
3. **导入配置**
4. **测试功能**

### 版本兼容性

import { Callout } from 'nextra/components'

<Callout type="info">
  **兼容性说明**
  
  - 主版本号变更（如 3.x → 4.x）可能不兼容
  - 次版本号变更（如 4.4 → 4.5）通常兼容
  - 修订版本（如 4.5.0 → 4.5.1）完全兼容
</Callout>

<Callout type="warning">
  **升级注意**
  
  - 重大版本升级前务必备份
  - 阅读迁移指南
  - 在测试环境验证
  - 准备回滚方案
</Callout>

## 🔄 降级指南

### 何时需要降级

- 新版本存在严重 bug
- 项目不兼容新版本
- 性能问题
- 功能缺失

### 降级步骤

1. **停止使用新版本**
2. **恢复备份**
   ```bash
   rm -rf WebGAL_current
   cp -r WebGAL_backup WebGAL_current
   ```
3. **验证项目**
4. **报告问题**
