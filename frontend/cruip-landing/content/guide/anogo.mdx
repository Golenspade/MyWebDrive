# AnoGO 工具使用指南

AnoGO 是一个强大的资源管理与打包工具，帮助你高效管理视觉小说项目的各类资源。

## 📥 下载与安装

### 获取软件

前往 [下载中心](/download) 下载最新版本的 AnoGO。

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+, Linux
- **内存**: 至少 2GB RAM
- **存储**: 1GB 可用空间
- **依赖**: Node.js 16+ (可选)

## 🎯 主要功能

### 资源管理

- **批量导入**: 一次性导入多个资源文件
- **分类整理**: 自动按类型分类资源
- **重命名工具**: 批量重命名文件
- **去重检测**: 识别并删除重复资源

### 资源优化

- **图片压缩**: 无损或有损压缩图片
- **音频转换**: 转换音频格式和码率
- **视频压缩**: 优化视频文件大小
- **批量处理**: 一键优化所有资源

### 打包功能

- **项目打包**: 将项目打包为发布版本
- **增量更新**: 生成更新补丁包
- **多平台支持**: 针对不同平台优化
- **加密保护**: 可选资源加密

### 版本控制

- **版本管理**: 记录资源版本历史
- **回滚功能**: 恢复到之前的版本
- **变更追踪**: 查看资源修改记录
- **备份还原**: 自动备份重要资源

## 📖 使用教程

### 创建新项目

import { Steps } from 'nextra/components'

<Steps>

### 启动 AnoGO

打开 AnoGO 工具。

### 新建项目

点击"文件" → "新建项目"。

### 设置项目信息

填写项目名称和保存路径。

### 导入资源

将资源文件拖入或点击导入。

</Steps>

### 批量导入资源

1. 点击"导入" → "批量导入"
2. 选择资源文件夹
3. 选择导入选项
   - 保持原始结构
   - 自动分类
   - 重命名规则
4. 确认导入

### 资源优化

#### 图片优化

```bash
# 压缩所有 PNG 图片
anogo optimize --type=image --format=png --quality=85

# 转换为 WebP 格式
anogo convert --from=png --to=webp --quality=90
```

#### 音频优化

```bash
# 转换音频格式
anogo convert --from=wav --to=mp3 --bitrate=192k

# 批量降低码率
anogo optimize --type=audio --bitrate=128k
```

### 项目打包

1. 点击"打包" → "创建发布包"
2. 选择目标平台
   - Web 版本
   - Windows 版本
   - macOS 版本
   - Linux 版本
3. 配置打包选项
   - 压缩级别
   - 是否加密
   - 包含文件
4. 开始打包

## 💡 高级功能

### 自动化脚本

创建自动化工作流：

```javascript
// workflow.js
module.exports = {
  tasks: [
    {
      name: '优化图片',
      action: 'optimize',
      type: 'image',
      options: { quality: 85 }
    },
    {
      name: '转换音频',
      action: 'convert',
      type: 'audio',
      options: { format: 'mp3', bitrate: '192k' }
    },
    {
      name: '打包项目',
      action: 'package',
      platform: 'web'
    }
  ]
};
```

运行工作流：

```bash
anogo run workflow.js
```

### 插件系统

安装插件扩展功能：

```bash
# 安装图片优化插件
anogo plugin install image-optimizer

# 安装资源分析插件
anogo plugin install resource-analyzer
```

### 命令行工具

```bash
# 查看项目信息
anogo info

# 列出所有资源
anogo list --type=all

# 查找重复文件
anogo find-duplicates

# 生成资源报告
anogo report --output=report.html
```

## 📊 资源分析

### 生成分析报告

```bash
anogo analyze --output=analysis.json
```

报告内容包括：

- 资源总数和大小
- 各类型资源占比
- 重复文件列表
- 优化建议

### 性能优化建议

- **图片**: 使用 WebP 格式，压缩质量 80-90
- **音频**: MP3 192kbps 或 OGG 128kbps
- **视频**: H.264 编码，适中码率
- **总体**: 控制总包大小在合理范围

## 🆘 常见问题

### 导入失败

- 检查文件格式是否支持
- 确认文件没有损坏
- 查看错误日志

### 优化后质量下降

- 提高压缩质量参数
- 使用无损压缩模式
- 对比原始文件

### 打包错误

- 检查磁盘空间
- 确认所有资源完整
- 查看打包日志
