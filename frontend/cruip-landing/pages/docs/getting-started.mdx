# 快速入门

欢迎使用 MyGO 视觉小说创作平台！本指南将帮助你快速开始创作之旅。

## 🎯 平台概述

MyGO 是一个一站式视觉小说创作平台，提供：

- **完整工具链** - 从脚本编写到发布的全流程支持
- **跨平台分发** - 一次创作，多端发布
- **资源共享** - 海量素材与改模资源
- **社区驱动** - 活跃的创作者生态

## 📋 准备工作

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+, Linux
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+
- **存储空间**: 至少 2GB 可用空间
- **网络**: 稳定的互联网连接

### 账号注册

1. 访问 [注册页面](/signup)
2. 填写邮箱和密码
3. 验证邮箱（如需要）
4. 完成个人资料设置

## 🚀 三步开始创作

### 第一步：选择引擎

根据你的需求选择合适的引擎：

- **WebGAL 标准版** - 适合大多数项目，功能全面
- **WebGAL MyGO 专版** - 针对 MyGO 主题优化
- **东山定制版** - 特殊功能定制版本

前往 [下载中心](/download) 获取引擎。

### 第二步：下载工具

根据项目需求下载配套工具：

- **Live2D Watcher** - Live2D 模型预览与调试
- **AnoGO** - 资源管理与打包工具
- **Bandori Craft** - 邦邦主题素材编辑器

查看 [工具使用指南](/docs/guide) 了解详情。

### 第三步：创建项目

1. 启动 WebGAL 引擎
2. 创建新项目
3. 编写脚本或导入模板
4. 添加素材（图片、音频、视频）
5. 预览和调试
6. 导出发布
