# Live2D Watcher 使用指南

Live2D Watcher 是一个专业的 Live2D 模型预览与调试工具，帮助你轻松管理和优化 Live2D 资源。

## 📥 下载与安装

### 获取软件

前往 [下载中心](/download) 下载最新版本的 Live2D Watcher。

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+
- **显卡**: 支持 OpenGL 3.3+
- **内存**: 至少 4GB RAM
- **存储**: 500MB 可用空间

## 🎯 主要功能

### 模型预览

- **实时渲染**: 即时查看模型效果
- **多角度查看**: 360° 旋转预览
- **背景切换**: 测试不同场景下的效果
- **缩放调整**: 自由缩放模型大小

### 动作调试

- **动作列表**: 查看所有可用动作
- **动作播放**: 单独播放测试
- **速度调节**: 调整播放速度
- **循环设置**: 设置循环播放

### 表情管理

- **表情切换**: 快速切换表情
- **表情组合**: 测试表情叠加效果
- **参数调节**: 微调表情参数
- **预设保存**: 保存常用表情

### 导出功能

- **截图导出**: 导出高质量预览图
- **GIF 录制**: 录制动画 GIF
- **视频导出**: 导出 MP4 视频
- **参数导出**: 导出配置文件

## 📖 使用教程

### 加载 Live2D 模型

1. 点击"文件" → "打开模型"
2. 选择 `.model3.json` 文件
3. 等待模型加载完成
4. 模型显示在预览窗口

### 调试模型动作

1. 在动作面板选择动作
2. 点击播放按钮
3. 观察动作效果
4. 调整参数优化

### 导出预览图

1. 调整模型到理想状态
2. 点击"导出" → "截图"
3. 选择分辨率和格式
4. 保存到指定位置

### 批量处理

1. 加载多个模型
2. 设置统一参数
3. 批量导出预览图
4. 自动命名保存

## 🔗 集成到 WebGAL

### 导出适配

1. 导出 PNG 格式预览图
2. 调整到合适尺寸
3. 放入 WebGAL 项目的 `figure/` 目录
4. 在脚本中引用

### 最佳实践

- **分辨率**: 建议 1000-2000px 高度
- **格式**: PNG 透明背景
- **命名**: 使用规范的文件名
- **优化**: 压缩文件大小

## 🆘 故障排除

### 模型无法加载

- 检查文件路径是否正确
- 确认模型文件完整
- 查看错误日志

### 性能问题

- 降低渲染质量
- 关闭不必要的特效
- 更新显卡驱动

### 导出失败

- 检查磁盘空间
- 确认导出路径权限
- 尝试降低导出质量
