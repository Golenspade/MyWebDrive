# Bandori Craft 使用指南

Bandori Craft 是专为 BanG Dream! 主题视觉小说设计的素材编辑器，提供丰富的邦邦主题资源和编辑功能。

## 📥 下载与安装

### 获取软件

前往 [下载中心](/download) 下载最新版本的 Bandori Craft。

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+
- **内存**: 至少 4GB RAM
- **存储**: 2GB 可用空间
- **显卡**: 支持硬件加速

## 🎯 主要功能

### 角色立绘编辑

- **立绘库**: 内置所有邦邦角色立绘
- **表情编辑**: 自定义角色表情
- **服装切换**: 更换角色服装
- **姿势调整**: 调整角色姿势
- **特效添加**: 添加光效、粒子等

### 背景素材制作

- **场景库**: 丰富的邦邦场景背景
- **自定义背景**: 导入和编辑背景
- **图层管理**: 多图层编辑
- **滤镜效果**: 各种视觉效果
- **动态背景**: 创建动画背景

### UI 元素定制

- **对话框**: 自定义对话框样式
- **按钮**: 设计交互按钮
- **菜单**: 创建游戏菜单
- **HUD**: 设计界面元素
- **主题包**: 完整 UI 主题

### 资源导出

- **批量导出**: 一键导出所有资源
- **格式转换**: 支持多种格式
- **尺寸调整**: 自动适配分辨率
- **打包功能**: 生成资源包

## 📖 使用教程

### 创建角色立绘

import { Steps } from 'nextra/components'

<Steps>

### 选择角色

从资源库选择邦邦角色。

### 选择服装

选择角色的服装版本。

### 调整表情

在表情面板选择或自定义表情。

### 添加特效

可选添加光效、粒子等特效。

### 导出立绘

导出为 PNG 格式，透明背景。

</Steps>

### 制作对话框

1. 点击"UI 元素" → "对话框"
2. 选择对话框模板
3. 自定义样式
   - 背景颜色/图片
   - 边框样式
   - 文字样式
4. 预览效果
5. 导出为图片或 CSS

### 创建主题包

1. 新建主题项目
2. 设计各个 UI 元素
   - 对话框
   - 按钮
   - 菜单
   - 背景
3. 统一配色方案
4. 导出主题包

## 🔧 高级功能

### 动画编辑

创建角色动画：

1. 在时间轴添加关键帧
2. 调整角色位置/表情
3. 设置过渡效果
4. 预览动画
5. 导出为 GIF 或视频

### 脚本集成

生成 WebGAL 脚本：

```javascript
// 自动生成的脚本
changeFigure:character_happy.png -position=center;
changeBg:school_classroom.jpg;
playBgm:bgm_daily.mp3;
```

### 批量处理

```bash
# 批量调整尺寸
Tools → Batch → Resize All

# 批量添加水印
Tools → Batch → Add Watermark

# 批量导出
Tools → Batch → Export All
```

## 🔗 集成到项目

### 导出到 WebGAL

1. 导出所有资源到项目文件夹
2. 资源自动分类到对应目录
   - 立绘 → `figure/`
   - 背景 → `background/`
   - UI → `ui/`
3. 生成资源清单
4. 可选生成脚本模板

### 文件命名规范

```
# 角色立绘
character_name_emotion_costume.png
例: kasumi_happy_default.png

# 背景
scene_location_time.jpg
例: school_classroom_day.jpg

# UI 元素
ui_element_type_state.png
例: button_start_normal.png
```

## 🆘 常见问题

### 资源无法导入

- 检查文件格式（支持 PNG, JPG, PSD）
- 确认文件大小不超过限制
- 查看错误提示

### 导出质量问题

- 提高导出质量设置
- 使用 PNG 格式保持透明度
- 检查原始资源质量

### 性能卡顿

- 关闭不必要的图层
- 降低预览质量
- 清理缓存文件
