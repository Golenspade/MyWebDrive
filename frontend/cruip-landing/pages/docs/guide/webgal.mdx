# WebGAL 引擎使用指南

WebGAL 是一个功能强大的视觉小说引擎，支持创建跨平台的视觉小说作品。

## 📥 下载与安装

### 版本选择

- **WebGAL 标准版** - 适合大多数项目
- **WebGAL MyGO 专版 3.0.0** - MyGO 主题优化版本
- **东山定制版** - 包含特殊功能

前往 [下载中心](/download) 获取最新版本。

## 🎯 快速开始

### 创建第一个项目

1. 启动 WebGAL 引擎
2. 点击"新建项目"
3. 输入项目名称和保存路径
4. 选择项目模板（可选）
5. 点击"创建"

### 项目结构

```
my-project/
├── game/
│   ├── scene/          # 场景脚本
│   ├── background/     # 背景图片
│   ├── figure/         # 角色立绘
│   ├── bgm/           # 背景音乐
│   ├── vocal/         # 语音文件
│   └── video/         # 视频文件
├── config.txt         # 项目配置
└── index.html         # 入口文件
```

## ✍️ 脚本编写

### 基础语法

#### 显示文本

```
这是一段对白。
```

#### 角色对话

```
角色名:这是角色说的话。
```

#### 显示背景

```
changeBg:background.jpg;
```

#### 显示立绘

```
changeFigure:character.png;
```

#### 播放音乐

```
playBgm:music.mp3;
```

### 高级功能

#### 分支选择

```
choose:选项A:sceneA|选项B:sceneB;
```

#### 变量系统

```
setVar:score=0;
setVar:score=score+10;
```

#### 条件判断

```
if:score>50;
  这是高分剧情。
else;
  这是低分剧情。
endif;
```

## 🎨 素材管理

### 图片素材

- **背景图片**: 推荐 1920x1080 或 1280x720
- **角色立绘**: PNG 格式，透明背景
- **UI 元素**: 根据设计需求

### 音频素材

- **背景音乐**: MP3/OGG 格式，循环播放
- **音效**: 短音频，WAV/MP3 格式
- **语音**: 清晰的人声录音

### 视频素材

- **格式**: MP4 (H.264)
- **分辨率**: 与游戏分辨率一致
- **码率**: 适中，避免文件过大

## 🔧 配置选项

### config.txt 配置

```ini

Game_key:my-game
Title_img:title.png
Title_bgm:title.mp3
```

### 高级配置

- **字体设置**: 自定义字体样式
- **UI 主题**: 修改界面外观
- **性能优化**: 资源预加载设置

## 🚀 导出与发布

### Web 版本

1. 点击"导出" → "Web 版本"
2. 选择导出路径
3. 等待打包完成
4. 上传到服务器或托管平台

### 桌面版本

1. 点击"导出" → "桌面版本"
2. 选择目标平台（Windows/macOS/Linux）
3. 配置打包选项
4. 生成可执行文件

### 移动版本

1. 使用 Cordova/Capacitor 打包
2. 配置移动端适配
3. 生成 APK/IPA 文件
