# 东山工具集使用指南

东山工具集是一套专业的视觉小说开发辅助工具，提供高级功能和性能优化方案。

## 📥 下载与安装

### 获取工具集

前往 [下载中心](/download) 下载东山工具集完整包。

### 系统要求

- **操作系统**: Windows 10/11, macOS 10.15+, Linux
- **内存**: 至少 8GB RAM
- **存储**: 3GB 可用空间
- **处理器**: 多核处理器推荐

## 🛠️ 工具组件

### 特殊效果编辑器

创建和编辑高级视觉效果：

- **粒子系统**: 雨、雪、樱花等粒子效果
- **光影效果**: 动态光照、阴影、光晕
- **后处理**: 模糊、景深、色彩分级
- **转场效果**: 自定义场景转场动画
- **滤镜库**: 各种视觉滤镜

### 高级脚本工具

增强脚本编写体验：

- **代码补全**: 智能代码提示
- **语法检查**: 实时错误检测
- **脚本调试**: 断点调试功能
- **性能分析**: 脚本性能优化
- **版本对比**: 脚本版本管理

### 性能分析器

优化项目性能：

- **资源分析**: 检测资源使用情况
- **内存监控**: 实时内存占用
- **加载优化**: 优化加载速度
- **帧率监控**: 性能瓶颈定位
- **优化建议**: 自动生成优化方案

### 调试辅助工具

提升开发效率：

- **实时预览**: 即时查看修改效果
- **日志查看器**: 详细的运行日志
- **变量监控**: 监控游戏变量
- **事件追踪**: 追踪事件触发
- **错误定位**: 快速定位问题

### 性能分析

#### 运行分析

```bash
# 启动性能分析器
dongshan-analyzer --project=./my-project

# 生成报告
dongshan-analyzer --report --output=report.html
```

报告包含：

- **资源统计**: 各类资源数量和大小
- **加载时间**: 各资源加载耗时
- **内存占用**: 运行时内存使用
- **性能瓶颈**: 识别性能问题
- **优化建议**: 具体优化方案

### 性能优化建议

import { Callout } from 'nextra/components'

<Callout type="info">
  **优化策略**
  - 使用纹理图集减少绘制调用
  - 启用资源预加载
  - 合理使用对象池
  - 优化粒子系统参数
  - 使用 LOD（细节层次）技术
</Callout>

<Callout type="warning">
  **注意事项**
  - 过多的后处理效果会影响性能
  - 大量粒子会增加 CPU 负担
  - 实时光照计算较为耗时
  - 调试模式会降低运行速度
</Callout>

## 💡 实战技巧

### 优化加载速度

1. **资源预加载**
   ```javascript
   preload([
     'background/scene1.jpg',
     'figure/character1.png',
     'bgm/music1.mp3'
   ]);
   ```

2. **延迟加载**
   ```javascript
   lazyLoad('video/opening.mp4', () => {
     // 加载完成后播放
     playVideo('opening');
   });
   ```

3. **资源压缩**
   - 使用 WebP 格式图片
   - 压缩音频文件
   - 启用 Gzip 压缩

### 调试技巧

```javascript
// 性能监控
const monitor = new PerformanceMonitor();
monitor.start();

// 场景代码
loadScene('chapter1');

// 获取性能数据
const metrics = monitor.stop();
console.log('FPS:', metrics.fps);
console.log('内存:', metrics.memory);
console.log('加载时间:', metrics.loadTime);
```

## 🔗 集成到项目

### 与 WebGAL 集成

1. 在 WebGAL 项目中启用东山工具集
2. 导入效果配置文件
3. 在脚本中引用效果
4. 测试和优化

### 导出配置

```bash
# 导出所有配置
dongshan export --all --output=./configs

# 导出特定效果
dongshan export --effect=sakura_fall --output=./effects

# 生成集成代码
dongshan generate --template=webgal
```

## 🆘 故障排除

### 效果不显示

- 检查配置文件格式
- 确认资源路径正确
- 查看浏览器控制台错误
- 验证 WebGL 支持

### 性能问题

- 降低粒子数量
- 减少后处理效果
- 优化光照数量
- 使用性能分析器定位问题

### 调试器无法连接

- 检查端口是否被占用
- 确认防火墙设置
- 验证配置文件
- 重启调试服务
