# 工具使用指南

欢迎来到工具使用指南！这里提供了平台所有工具的详细使用说明。

## 🎮 核心引擎

### WebGAL 引擎系列

WebGAL 是我们的核心视觉小说引擎，提供多个版本以满足不同需求：

- **[WebGAL 标准版](/docs/guide/webgal)** - 功能全面的标准引擎
- **WebGAL MyGO 专版** - 针对 MyGO 主题优化的特别版本
- **东山定制版** - 包含特殊功能的定制版本

[查看 WebGAL 完整教程 →](/docs/guide/webgal)

## 🛠️ 配套工具

### Live2D Watcher

Live2D 模型预览与调试工具，帮助你：

- 实时预览 Live2D 模型
- 调试模型动作和表情
- 导出预览图片
- 性能优化建议

[查看详细教程 →](/docs/guide/live2d-watcher)

### AnoGO 工具

资源管理与打包工具，提供：

- 资源批量管理
- 自动打包优化
- 版本控制
- 资源压缩

[查看详细教程 →](/docs/guide/anogo)

### Bandori Craft

邦邦主题素材编辑器，支持：

- 角色立绘编辑
- 背景素材制作
- UI 元素定制
- 主题包导出

[查看详细教程 →](/docs/guide/bandori-craft)

### 东山工具集

东山定制工具集合，包含：

- 特殊效果编辑器
- 高级脚本工具
- 性能分析器
- 调试辅助工具

[查看详细教程 →](/docs/guide/dongshan-tools)

## 📖 使用建议

### 新手推荐流程

1. **先学习 WebGAL** - 从核心引擎开始
2. **按需学习工具** - 根据项目需求选择工具
3. **实践练习** - 通过实际项目巩固知识
4. **参考最佳实践** - 学习优秀案例

### 工具组合建议

#### 基础组合
- WebGAL 标准版
- Live2D Watcher（如使用 Live2D）

#### 进阶组合
- WebGAL 标准版
- Live2D Watcher
- AnoGO 工具

#### 专业组合
- WebGAL MyGO 专版
- Live2D Watcher
- AnoGO 工具
- Bandori Craft
- 东山工具集
