# 示例与集成

这里提供常见用法的最小可用示例，方便你快速集成。

> Base URL: http://localhost:9080/api/v1

## cURL

### 登录获取 Token
```bash
curl -X POST "http://localhost:9080/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"secure_password"}'
```

### 查询文件夹列表
```bash
curl -H "Authorization: Bearer $ACCESS_TOKEN" \
  "http://localhost:9080/api/v1/folders?parentId=root"
```

## JavaScript (fetch)

```js
// 登录
const login = async () => {
  const res = await fetch("/api/v1/auth/login", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ email: "user@example.com", password: "secure_password" })
  })
  if (!res.ok) throw new Error("Login failed")
  return res.json() // { accessToken, refreshToken }
}

// 获取文件夹
const listFolders = async (token) => {
  const res = await fetch("/api/v1/folders?parentId=root", {
    headers: { Authorization: `Bearer ${token}` }
  })
  if (!res.ok) throw new Error("List folders failed")
  return res.json()
}
```

## TypeScript (axios)

```ts
import axios from "axios"

// 创建 axios 实例
const api = axios.create({ baseURL: "/api/v1" })

export async function login(email: string, password: string) {
  const { data } = await api.post("/auth/login", { email, password })
  api.defaults.headers.common.Authorization = `Bearer ${data.accessToken}`
  return data
}

export async function createFolder(name: string, parentId = "root") {
  const { data } = await api.post("/folders", { name, parentId })
  return data
}
```
