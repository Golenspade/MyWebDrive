# API 文档

欢迎使用 MyGO 平台 API 文档。这里提供完整的 API 参考和使用指南。

## 🎯 API 概述

### 什么是 API

API (Application Programming Interface) 允许你通过编程方式与平台交互，实现：

- 用户认证和授权
- 文件和资源管理
- 项目数据操作
- 自动化工作流
- 第三方集成

### API 特性

- **RESTful 设计**: 遵循 REST 架构风格
- **JSON 格式**: 统一使用 JSON 数据格式
- **JWT 认证**: 安全的令牌认证机制
- **版本控制**: 支持 API 版本管理
- **速率限制**: 防止滥用的限流机制

## 🚀 快速开始

### 基础 URL

```
https://api.mygoavemujica.top/api/v1
```

开发环境：
```
http://localhost:9080/api/v1
```

### 认证方式

使用 JWT (JSON Web Token) 进行认证：

```http
Authorization: Bearer YOUR_ACCESS_TOKEN
```

### 第一个 API 请求

```javascript
// 获取用户信息
fetch('http://localhost:9080/api/v1/users/me', {
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN',
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data));
```

## 📚 API 分类

### 认证 API

用户注册、登录和令牌管理：

- `POST /auth/register` - 用户注册
- `POST /auth/login` - 用户登录
- `POST /auth/refresh` - 刷新令牌
- `POST /auth/logout` - 用户登出

[查看认证 API 详情 →](/docs/api/authentication)

### 用户 API

用户信息和配置管理：

- `GET /users/me` - 获取当前用户信息
- `PATCH /users/me` - 更新用户信息
- `PATCH /users/me/password` - 修改密码
- `GET /users/me/quota` - 获取存储配额

### 文件 API

文件上传、下载和管理：

- `POST /files` - 上传文件
- `GET /files/:id` - 获取文件信息
- `GET /files/:id/download` - 下载文件
- `PATCH /files/:id` - 更新文件信息
- `DELETE /files/:id` - 删除文件

### 文件夹 API

文件夹结构管理：

- `POST /folders` - 创建文件夹
- `GET /folders/:id/children` - 获取子项
- `PATCH /folders/:id` - 更新文件夹
- `DELETE /folders/:id` - 删除文件夹
- `POST /folders/:id/move` - 移动文件夹

### 分享 API

文件分享和协作：

- `POST /shares` - 创建分享
- `GET /shares/:id` - 获取分享信息
- `DELETE /shares/:id` - 删除分享
- `GET /public/shares/:code` - 访问公开分享

### 目录 API

下载中心资源目录：

- `GET /catalog` - 获取资源列表
- `GET /catalog/:slug` - 获取资源详情

## 🔧 通用规范

### 请求格式

**Headers**
```http
Content-Type: application/json
Authorization: Bearer YOUR_ACCESS_TOKEN
X-Request-ID: unique-request-id (可选)
```

**Body (JSON)**
```json
{
  "field1": "value1",
  "field2": "value2"
}
```

### 响应格式

**成功响应**
```json
{
  "success": true,
  "data": {
    "id": "123",
    "name": "example"
  }
}
```

**错误响应**
```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "错误描述"
  }
}
```

### HTTP 状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 201 | 创建成功 |
| 204 | 删除成功（无内容） |
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 429 | 请求过于频繁 |
| 500 | 服务器错误 |

import { Steps } from 'nextra/components'

```javascript
POST /api/v1/auth/login
```

### 获取令牌

响应包含访问令牌和刷新令牌：
```json
{
  "accessToken": "eyJhbGc...",
  "refreshToken": "eyJhbGc...",
  "expiresIn": 3600
}
```

### 使用令牌

在后续请求中携带访问令牌：
```http
Authorization: Bearer eyJhbGc...
```

</Steps>

### 刷新令牌

访问令牌过期时使用刷新令牌获取新令牌：

```javascript
POST /api/v1/auth/refresh
{
  "refreshToken": "eyJhbGc..."
}
```

## 📊 速率限制

### 限制规则

- **未认证请求**: 100 次/小时
- **已认证请求**: 1000 次/小时
- **文件上传**: 100 次/小时
- **下载请求**: 500 次/小时

### 响应头

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640000000
```

### 超限处理

超过限制时返回 429 状态码：

```json
{
  "success": false,
  "error": {
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "请求过于频繁，请稍后再试",
    "retryAfter": 3600
  }
}
```

## 🛠️ SDK 和工具

### JavaScript/TypeScript SDK

```bash
npm install @mygo/api-client
```

```javascript
import { MyGOClient } from '@mygo/api-client';

const client = new MyGOClient({
  baseURL: 'http://localhost:9080/api/v1',
  accessToken: 'YOUR_ACCESS_TOKEN'
});

// 获取用户信息
const user = await client.users.me();
```

### Python SDK

```bash
pip install mygo-api
```

```python
from mygo import Client

client = Client(
    base_url='http://localhost:9080/api/v1',
    access_token='YOUR_ACCESS_TOKEN'
)

# 获取用户信息
user = client.users.me()
```
