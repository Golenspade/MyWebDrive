## 🎮 引擎下载

### WebGAL 标准版

**下载信息**
- 文件名: `WebGAL_Standard_vX.X.X.zip`
- 大小: 约 50MB
- 格式: ZIP 压缩包

**安装步骤**
1. 下载压缩包
2. 解压到任意目录
3. 运行启动脚本
4. 开始创作

## 🛠️ 工具下载

### Live2D Watcher

**下载信息**
- 文件名: `l2dw-1.4.21f1修复专注预览界面不换行不适配问题.7z`
- 大小: 约 100MB
- 格式: 7Z 压缩包

**系统要求**
- Windows 10/11 或 macOS 10.15+
- 支持 OpenGL 3.3+
- 至少 4GB RAM

**安装说明**
1. 下载 7Z 压缩包
2. 使用 7-Zip 或 WinRAR 解压
3. 运行可执行文件
4. 首次启动可能需要配置权限

### AnoGO 工具

**下载信息**
- 文件名: `AnoGO_vX.X.X.zip`
- 大小: 约 30MB
- 格式: ZIP 压缩包

**快速开始**
1. 解压到工作目录
2. 运行 `anogo.exe` (Windows) 或 `anogo` (macOS/Linux)
3. 创建或打开项目
4. 开始管理资源

### Bandori Craft

**下载信息**
- 文件名: `BandoriCraft_vX.X.X.zip`
- 大小: 约 150MB
- 格式: ZIP 压缩包

**注意事项**
- 首次启动需要下载资源库
- 建议预留 2GB 存储空间
- 需要稳定的网络连接

## 🔍 文件验证

### 验证文件完整性

使用 MD5/SHA256 校验：

```bash
# Windows (PowerShell)
Get-FileHash -Algorithm SHA256 filename.zip

# macOS/Linux
shasum -a 256 filename.zip
```

## 🌐 下载加速

### 使用 CDN

部分资源提供 CDN 加速：
- 国内 CDN: 阿里云 OSS
- 国际 CDN: Cloudflare

### 下载工具推荐

- **Windows**: Internet Download Manager (IDM)
- **macOS**: Folx
- **Linux**: aria2
- **跨平台**: Free Download Manager

### 命令行下载

```bash
# 使用 wget
wget https://example.com/resource.zip

# 使用 curl
curl -O https://example.com/resource.zip

# 使用 aria2 (多线程)
aria2c -x 16 https://example.com/resource.zip
```

## 🆘 下载问题

### 速度慢

- 更换下载时间
- 使用下载工具
- 尝试镜像站点
- 使用 VPN (国际资源)

### 无法下载

- 检查浏览器设置
- 禁用广告拦截器
- 清除 Cookie
- 更换浏览器

### 文件打不开

- 确认文件格式
- 安装解压工具
- 检查文件完整性
- 重新下载
